Basic Networking Commands

1. Finding the IP Address Using ip

The ip command is used to display and manipulate routing, network devices, interfaces, and tunnels.

ip addr show

This command lists all the network interfaces along with their IP addresses.

2. Checking Connectivity with ping

The ping command checks the connectivity between the host machine and a target domain or IP address.

ping google.com

Press Ctrl+C to stop pinging.

3. DNS and Name Resolution

DNS (Domain Name System) resolves domain names into IP addresses.

  • /etc/hosts: First checked for name resolutions.
  • /etc/resolv.conf: Lists DNS servers used for resolving domain names.

4. Managing DNS with systemd-resolved

Modern systems use systemd-resolved for managing DNS settings. It's configured through /etc/systemd/resolved.conf.

ls -l /etc/resolv.conf
cat /etc/resolv.conf

5. Using resolvectl for DNS Management

resolvectl is used to interact with systemd-resolved to manage DNS settings and view DNS statistics.

resolvectl status
resolvectl query example.com

6. Basic DNS Lookup with host

The host command performs DNS lookups and displays IP addresses for given hostnames.

host www.example.com

7. Advanced DNS Queries with dig

dig provides detailed information about domain names and their DNS records.

dig example.com
dig +short example.com TXT

8. Socket Statistics with ss

The ss command is used to dump socket statistics. It can display sockets, listening ports, and other details.

ss -tulwn

9. Tracing Network Route with traceroute

traceroute shows the path packets take to reach a host.

traceroute www.example.com

10. Tracing Path with tracepath

tracepath traces the path to a network host and discovers the MTU (Maximum Transmission Unit) along that path.

tracepath www.example.com

11. Remote Login with ssh

SSH (Secure Shell) is used for secure remote login from one computer to another.

ssh username@hostname

12. Generating SSH Keys with ssh-keygen

SSH keys enhance security for SSH connections.

ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

13. Copying SSH Keys with ssh-copy-id

This command is used to copy SSH public keys to remote hosts.

ssh-copy-id username@hostname

14. Disabling SSH Password Authentication

For increased security, password authentication can be disabled in SSH.

Edit /etc/ssh/sshd_config and set PasswordAuthentication to no.

15. Viewing Active Network Connections with lsof

List open files related to network connections using lsof.

sudo lsof -i
