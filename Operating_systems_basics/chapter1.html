Introduction to Shell Scripting - Chapter 1 and more

Shell scripting is a powerful tool for automating tasks in Unix-like operating systems. It allows you to combine multiple commands, logic, and utilities into a single script, which can be executed to perform complex operations efficiently. This tutorial will guide you through the basics of shell scripting, covering essential commands and concepts.

Getting Started

Displaying Date and Time

  1. Current Date and Time

    The date command displays the current date and time.

    bash date

  2. Yesterday's Date

    To display the date for yesterday, use:

    bash date --date="yesterday"

  3. Date 10 Days Ago

    To find out what the date was 10 days ago:

    bash date --date="10 days ago"

  4. Calendar

    The cal command shows the calendar for the current month.

    bash cal

User Information

  1. Current User

    To display the username of the user executing the script:

    bash whoami

  2. User and Group IDs

    The id command shows the user ID and group ID.

    bash id

Navigating Directories

  1. Current Working Directory

    Use pwd to print the current working directory.

    bash pwd

  2. Changing Directories

    • To change to the /home directory:

      bash cd /home

    • To move to the parent directory:

      bash cd ..

    • To switch to the user's home directory:

      bash cd ~

Managing Files and Directories

  1. Creating a Directory

    To create a new directory named "operating":

    bash mkdir operating

  2. Listing Contents

    The ls command lists the contents of a directory.

    bash ls

    For a detailed view, use ls -l.

  3. Creating and Viewing Files

    • To print "hello" to a file named "hello.txt":

      bash echo hello > hello.txt

    • Use cat to display the contents of "hello.txt":

      bash cat hello.txt

  4. Copying Files

    To copy "hello.txt" to "hello2.txt":

    bash cp hello.txt hello2.txt

Advanced Operations

  1. Piping and Redirection

    Piping (|) allows you to use the output of one command as the input for another.

    bash ls | wc -w

    Redirection (>) is used to send the output of a command to a file.

    bash ls > list.txt

  2. Removing Files and Directories

    • To remove a file:

      bash rm <filename>

    • To remove a directory and its contents:

      bash rm -r <directory>

  3. Using tree to Display Directory Structure

    The tree command visually displays the directory structure.

    bash tree

  4. Word and Line Counts

    • To count lines in a file:

      bash wc -l <file>

    • To count words:

      bash wc -w <file>

This tutorial covers fundamental shell scripting commands and concepts, aiming to provide a solid foundation for automating tasks in Unix-like environments. As you become more comfortable with these basics, you'll be well-prepared to explore more complex scripting techniques and utilities.
