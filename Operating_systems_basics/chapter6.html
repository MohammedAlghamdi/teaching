Chapter 6 - Processes in Linux

1. Viewing All Running Processes

Command: ps aux

Description: Lists all running processes on your system.

Example:

ps aux

2. Finding a Specific Process

Command: ps aux | grep [keyword]

Description: Filters the list of processes to only show those that match the keyword.

Example:

ps aux | grep firefox

3. Killing a Process

Command: kill [PID]

Description: Sends a termination signal to the process with the specified Process ID (PID).

Example:

kill 1234

4. Forcibly Killing a Process

Command: kill -9 [PID]

Description: Forcefully terminates the process by sending it the KILL signal.

Example:

kill -9 1234

5. Listing All Available Signals

Command: kill -l

Description: Lists all the signal names and numbers available for use with kill.

Example:

kill -l

6. Listing Open Files

Command: lsof

Description: Shows all open files and the processes that have opened them.

Example:

lsof

7. Monitoring System Processes

Command: top

Description: Provides a dynamic real-time view of all running system processes.

Example:

top

8. Interactive Process Viewer

Command: htop

Description: An interactive process viewer that offers a more user-friendly and detailed view than top.

Example:

sudo apt-get install htop # First install htop if not present
htop

9. Accessing Command Manuals

Command: man [command]

Description: Displays the manual page for the specified command.

Example:

man ls

10. Getting More Information on a Topic

Command: info [topic]

Description: Provides detailed documentation on the specified topic.

Example:

info libc

11. Viewing System Information Files

Command: cat /proc/[filename]

Description: Displays the contents of system information files located in the /proc directory.

Example:

cat /proc/cpuinfo

12. Viewing All System Settings

Command: sysctl -a

Description: Lists all kernel parameters and their current values.

Example:

sysctl -a

13. Modifying Kernel Parameters Temporarily

Command: sysctl -w [parameter=value]

Description: Temporarily changes a kernel parameter.

Example:

sysctl -w net.ipv4.ip_forward=1

14. Viewing a Kernel Parameter

Command: sysctl [parameter]

Description: Displays the current value of a specific kernel parameter.

Example:

sysctl net.ipv4.ip_forward

15. Checking IP Forwarding Status

Command: cat /proc/sys/net/ipv4/ip_forward

Description: Displays whether IP forwarding is enabled (1) or disabled (0).

Example:

cat /proc/sys/net/ipv4/ip_forward

16. Directly Modifying System Parameters

Command: echo [value] > /proc/sys/[path]

Description: Writes a value directly to a system parameter.

Example:

echo 1 > /proc/sys/net/ipv4/ip_forward

17. Applying System Configuration Changes

Command: sudo sysctl -p /etc/sysctl.conf

Description: Reloads and applies system configuration settings from the /etc/sysctl.conf file.

Example:

sudo sysctl -p /etc/sysctl.conf
