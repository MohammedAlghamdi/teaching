Week 12a: Understanding Functions and Procedures

This tutorial explains the basic concepts, syntax, and examples of functions and procedures in PL/SQL, making it easier for beginners to get started with database programming.

What is a Function in PL/SQL?

A function is a named block of code that performs a task and returns a value. It is useful for tasks that need to be repeated with different inputs and a result is expected every time.

Function Syntax

CREATE [OR REPLACE] FUNCTION function_name [parameters]
RETURN return_type
IS
-- Declaration section for variables
BEGIN
-- Code for processing
RETURN value;
EXCEPTION
-- Exception handling
END;

Example: Adding Two Numbers

CREATE FUNCTION add_numbers(x NUMBER, y NUMBER)
RETURN NUMBER
IS
result NUMBER;
BEGIN
result := x + y; -- Calculate the sum of x and y
RETURN result; -- Return the calculated sum
END;

Usage Example:

SELECT add_numbers(10, 20) FROM dual;

This SQL query will output 30.

What is a Procedure in PL/SQL?

A procedure is similar to a function but does not return a value. It's used to perform an operation, such as updating data or handling transactions.

Procedure Syntax

CREATE [OR REPLACE] PROCEDURE procedure_name [parameters]
IS
-- Declaration section for variables
BEGIN
-- Code for processing
EXCEPTION
-- Exception handling
END;

Example: Updating Employee Salary

CREATE PROCEDURE update_salary(emp_id NUMBER, new_salary NUMBER)
IS
BEGIN
UPDATE emp_tbl SET salary = new_salary WHERE empID = emp_id;
COMMIT;
EXCEPTION
WHEN OTHERS THEN
ROLLBACK;
RAISE;
END;

Usage Example:

EXECUTE update_salary(101, 65000);

This command updates the salary of employee 101 to 65000.

This tutorial has covered the basics of creating and using functions and procedures in PL/SQL. With practice, these tools can greatly enhance your ability to manipulate and manage data efficiently in an Oracle database.
