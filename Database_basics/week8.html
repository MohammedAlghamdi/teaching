Week 8: UNION, INTERSECT, MINUS

Suppose we have two tables

Employees Table

EmployeeIDNameDepartment
1AliceHR
2BobMarketing
3CharlieIT

Contractors Table

EmployeeIDNameDepartment
4DianaIT
5EricMarketing
3CharlieIT

SQL Operations

UNION

SELECT * FROM Employees
UNION
SELECT * FROM Contractors;

Result:

EmployeeIDNameDepartment
1AliceHR
2BobMarketing
3CharlieIT
4DianaIT
5EricMarketing

INTERSECT

SELECT * FROM Employees
INTERSECT
SELECT * FROM Contractors;

Result:

EmployeeIDNameDepartment
3CharlieIT

MINUS (or EXCEPT)

SELECT * FROM Employees
EXCEPT
SELECT * FROM Contractors;

Result:

EmployeeIDNameDepartment
1AliceHR
2BobMarketing
