SQL Syntax Explanations

Data Definition Language (DDL) Commands:

CREATE TABLE table_name (column_name data_type(size), column_name data_type(size));

This command is used to create a new table in the database, defining each column's name, data type, and size.

CREATE TABLE TABLE_NAME AS SELECT column_name FROM table_name;

This creates a new table by selecting columns from an existing table.

Data Manipulation Language (DML) Commands:

INSERT INTO table_name (column_name) VALUES (value);

This inserts new data into the table.

SELECT column_name FROM table_name;

This selects data from a table.

UPDATE table_name SET column_name = value WHERE condition;

This updates data in a table based on a condition.

DELETE FROM table_name WHERE condition;

This deletes data from a table based on a condition.

Constraints and Keys:

CREATE TABLE table_name (column_name data_type PRIMARY KEY);

This sets a column as the primary key, uniquely identifying each record in the table.

CREATE TABLE table_name (column_name data_type REFERENCES another_table(column_name));

This creates a foreign key that links to another table's primary key.

CREATE TABLE table_name (column_name data_type CHECK (condition));

This creates a constraint that only allows data which satisfies the condition.

Altering Table Structure:

ALTER TABLE table_name ADD column_name data_type;

This adds a new column to the table.

ALTER TABLE table_name DROP COLUMN column_name;

This removes a column from the table.

ALTER TABLE table_name MODIFY column_name new_data_type;

This changes the data type of a column.

Sorting of Data in a Table:

SELECT column_name FROM table_name ORDER BY column_name ASC|DESC;

This sorts the selected data in ascending or descending order.
