Week 11 : Functionalities of DBMS

SQL functions allow for manipulation of data directly in your queries, useful for processing strings, dates, and performing calculations. Below, we cover various SQL functions and provide examples of their use in a hypothetical Employees table.

1. String Manipulation Functions

  • CONCAT(): Joins two or more strings into one.
  • UPPER(), LOWER(), INITCAP(): Changes string casing.
  • LPAD(), RPAD(): Adds padding characters to a string.
  • LTRIM(), RTRIM(): Trims characters from a string.
  • REPLACE(), SUBSTR(), INSTR(): Modifies or inspects parts of a string.

2. Date Manipulation Functions

  • ADD_MONTHS(), NEXT_DAY(): Manipulates date values by adding months or finding the next specified weekday.
  • MONTHS_BETWEEN(): Calculates the number of months between two dates.
  • ROUND() (date): Rounds a date to the specified unit.

3. Conversion and Miscellaneous Functions

  • TO_CHAR(), TO_DATE(): Converts between strings and date or numeric formats.
  • ASCII(), CHR(): Converts between characters and their ASCII codes.
  • TRANSLATE(): Replaces several characters in a string based on a mapping.

Example SQL Setup and Queries

Assume we have an Employees table with employee details. Below are SQL commands to create this table, insert data, and apply SQL functions.

CREATE TABLE Employees ( employee_id INT, last_name VARCHAR(100), salary DECIMAL(10, 2), hire_date DATE ); INSERT INTO Employees VALUES (1, 'Baer', 50000, '1993-06-07'); INSERT INTO Employees VALUES (2, 'Smith', 40000, '2001-02-20'); INSERT INTO Employees VALUES (3, 'Jones', 55000, '1999-11-15');

Using String Functions

SELECT CONCAT(last_name, ' works here.') AS EmployeeStatement FROM Employees;
SELECT UPPER(last_name) AS UpperCaseLastName FROM Employees;
SELECT LPAD(last_name, 10, '*') AS PaddedLastName FROM Employees;

Using Date Functions

SELECT TO_CHAR(ADD_MONTHS(hire_date, 1), 'DD-MON-YYYY') AS NextMonth FROM Employees WHERE employee_id = 1;
SELECT MONTHS_BETWEEN('2024-04-01', hire_date) AS MonthsWorking FROM Employees;

Using Conversion Functions

SELECT ASCII(last_name) AS LastNameASCII FROM Employees WHERE last_name = 'Baer';
SELECT TO_CHAR(SYSDATE, 'Day, Month DD, YYYY') AS CurrentDate FROM DUAL;
