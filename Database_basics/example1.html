Tutorial Example 1

Creating a Table

SQL Command:

CREATE TABLE customers (
    customer_id INT PRIMARY KEY,
    name VARCHAR(100),
    age INT,
    email VARCHAR(255)
);

Expected Outcome: A new table named customers is created.

Inserting Multiple Rows into Table

SQL Command:

INSERT INTO customers (customer_id, name, age, email) VALUES
(1, 'John Doe', 30, 'john.doe@example.com'),
(2, 'Jane Doe', 25, 'jane.doe@example.com'),
(3, 'Alice Johnson', 28, 'alice.j@example.com'),
(4, 'Bob Smith', 32, 'bob.smith@example.com');

Expected Outcome: Four new rows are added to the customers table.

Selecting Data from Table

SQL Command:

SELECT * FROM customers;

Expected Outcome: Displays all rows from the customers table.

Updating Data for Multiple Rows

SQL Command:

UPDATE customers SET age = age + 1 WHERE age < 30;

Expected Outcome: Ages of customers under 30 are incremented by 1.

Deleting a Specific Row

SQL Command:

DELETE FROM customers WHERE customer_id = 2;

Expected Outcome: The row with customer_id 2 (Jane Doe) is deleted.

Adding a New Column for Customer Address

SQL Command:

ALTER TABLE customers ADD address VARCHAR(255);

Expected Outcome: A new column named address is added to the customers table.

Inserting New Data with Address

SQL Command:

INSERT INTO customers (customer_id, name, age, email, address) VALUES
(5, 'Eve Adams', 29, 'eve.adams@example.com', '123 Apple St.'),
(6, 'Gary White', 35, 'gary.white@example.com', '456 Orange Ave.');

Expected Outcome: Two new rows are added, including addresses.

Sorting Data by Age

SQL Command:

SELECT * FROM customers ORDER BY age DESC;

Expected Outcome: Displays all customers, sorted by age in descending order.

This expanded tutorial provides a comprehensive view of SQL commands and their practical application, enhancing the demonstration of SQL capabilities for database management and manipulation.
