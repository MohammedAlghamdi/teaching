Week 7: Group and Having

Step 1: Creating the Table

Create an SQL table named employee with five columns including one for the country and another for the salary. Here is the SQL command:

CREATE TABLE employee ( employee_id INT PRIMARY KEY, name VARCHAR(100), department VARCHAR(100), country VARCHAR(50), salary DECIMAL(10, 2) );

Step 3: Calculating Total Salaries by Country

To view the sum of salaries for each country, use the following query:

SELECT country, SUM(salary) AS total_salary FROM employee GROUP BY country;

Step 4: Filtering by Minimum Salary Total

To display countries where the total salary is 30,000 or more, use this query:

SELECT country, SUM(salary) AS total_salary FROM employee GROUP BY country HAVING SUM(salary) >= 30000;
